# Let's Work Together (LWT)

**LWT is an AI-powered Professional Work Execution Platform.**

Its purpose is to bridge the execution gap between education and employment by simulating realistic workplace environments using AI. The platform emotionally makes users feel "I work inside a real company" rather than "I am taking another online course."

## Overview

LWT replaces traditional learning management systems (LMS), coding bootcamps, and chatbots by providing an execution-first environment. Users join simulated AI organizations, collaborate with AI counterparts (AI CEO, PM, Tech Lead), and execute sprints exactly as they would in an enterprise tech environment.

## Key Features
- **AI Workforce:** Role-specific AI personalities that evaluate code, design, and execution.
- **Enterprise Emulation:** Join simulated organizations (SaaS, FinTech, Healthcare, etc.).
- **Sprint & Task Execution:** Professional workflows, standups, and performance tracking.
- **Portfolio Generation:** Automated documentation of real execution metrics.

## Tech Stack
- Next.js (React 19, Server Components)
- TypeScript
- Tailwind CSS & shadcn/ui
- Supabase & PostgreSQL
- Clerk Auth
- Gemini & OpenAI APIs
