# Analytics

## Product Analytics (PostHog)
We use PostHog to track user behavior without compromising privacy.
- **Key Metrics Tracked**:
  - Time to complete onboarding.
  - Organization selection rates (e.g., do users prefer FinTech or EdTech?).
  - Sprint completion velocity.
  - Drop-off points in the AI evaluation process.

## Data Privacy
- We do not send PII (Personally Identifiable Information) to PostHog unnecessarily. User IDs are hashed or anonymized where possible.
- Analytics are strictly used to improve the AI interaction and project realism, not for ad targeting.
