# User Journey & Workflow

The user experience in LWT simulates a complete job lifecycle, from onboarding to promotion.

## 1. Professional Onboarding
- **Landing Page**: Premium presentation of the platform.
- **Authentication**: Secure login via Clerk.
- **Career Profile**: User fills out their current skill level, target role (e.g., Full Stack Engineer), and preferences.

## 2. Joining the Company
- **Company Selection**: User browses simulated AI companies (SaaS, FinTech, Healthcare, etc.).
- **The Offer Letter**: User receives a formal, AI-generated offer letter detailing their role, salary (XP/Gamification), and expectations.
- **Meet the Team**: Introduction to their AI Manager, AI Tech Lead, and AI teammates.

## 3. The Daily Grind (Sprint Execution)
- **Sprint Planning**: AI PM assigns a sprint backlog. Dependencies are established.
- **Daily Standups**: User submits a quick update on blockers and progress.
- **Task Execution**: User works in their IDE and submits PRs/links/code to the platform.
- **AI Code Review**: AI Tech Lead provides strict, professional feedback. The user must iterate and fix issues.

## 4. Evaluation & Growth
- **Performance Evaluation**: End of sprint reviews focusing on code quality, communication, and speed.
- **Portfolio Generation**: The platform automatically updates the user's public portfolio with verified skills, sprint metrics, and AI recommendations.
- **Career Readiness**: User levels up, gaining access to harder projects or senior roles (e.g., moving from Junior Dev to System Architect).
