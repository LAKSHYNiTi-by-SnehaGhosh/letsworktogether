# AI Mentor & Career Coach

## Concept
While the AI Tech Lead is strict and focused on the company's codebase, the **AI Mentor** is focused on the user's personal career growth.

## Role
- Available 24/7 for 1-on-1 chats.
- Has access to the user's entire history across all simulated organizations.
- Helps explain *why* the AI Tech Lead rejected a PR without giving away the direct answer.
- Recommends which skills the user should focus on based on their Portfolio metrics.

## Implementation (Phase 4)
This requires a distinct chat interface (the only place a "chatbot" UI is acceptable in LWT), utilizing OpenAI for fast, empathetic responses, backed by a vector database of the user's past evaluations.
