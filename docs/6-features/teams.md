# AI Teams & Collaboration

## Concept
Users interact with distinct AI personas, not a generic "Assistant."

## Personas
- **AI CEO**: Focuses on business metrics and deadlines.
- **AI Product Manager (PM)**: Creates tasks, manages the backlog, cares about user experience.
- **AI Tech Lead**: Reviews code, cares about architecture, SOLID principles, and performance.
- **AI QA Engineer**: Tries to find edge cases and bugs in submitted work.

## Implementation
Each persona uses a specific prompt template injected with the current context of the project and user's history. They communicate through the platform's notification system and comment threads on tasks, simulating Slack/Teams or GitHub PR reviews.
