# Enterprise & University Deployment

## Vision
LWT is uniquely positioned to be sold B2B to coding bootcamps and universities that struggle with job placement rates.

## Custom Environments
- Enterprise admins can create a "sandbox" organization.
- They can force specific technologies into the AI's requirements (e.g., "This semester is about Java, generate a Spring Boot sprint").

## Admin Dashboard
Enterprise users get access to a separate route `/(admin)` which aggregates data across all their sponsored users, showing average task completion time and most common code review failures.
